# Track & Tote

Hybrid editorial plus data-aware personal site built with Astro and Markdown.

## Stack

- Astro
- MDX content collections
- RSS feed

## Commands

```bash
npm install
npm run dev
npm run build
npm run preview
```

## Content workflow

Write new posts in `src/content/posts/*.md` with frontmatter:

```yaml
---
title: Post title
excerpt: One-line summary
date: 2026-06-01
tags:
  - policy
  - data
draft: false
---
```
